# AI Mail 🚀

AI Mail is a full-stack AI-powered email generation and sending platform built using Rust and Next.js.

## Features

* AI Email Generation
* Send Emails using SMTP
* Inbox UI
* Sent Emails
* Email History
* Dark Professional UI
* PostgreSQL Database
* Docker Support

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Rust
* Axum
* SQLx
* PostgreSQL

### Other Tools

* Docker
* GitHub
* Groq API

## Project Structure

/backend → Rust Backend
/frontend → Next.js Frontend
/docker-compose.yml → Docker Configuration

## Run Locally

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cargo run
```
